<b>Salamandra</b> es un software de automatización de radio para <b>Windows</b>.

</p>

Diseñado para emisoras de radio que necesitan un software <b>sencillo</b>, <b>fácil</b> y <b>estable</b> para la transmisión automatizada.
</p>

<hr>

<p align="center">
Compatible con Windows 7 SP1 + ESE (*), 8.1, 10 y 11.
</p>

(*) Tu Windows 7 debe estar <strong>completamente actualizado</strong> (con Service Pack 1 + Actualizaciones de seguridad extendidas). Si tienes problemas con Windows Update, busca cómo solucionar el error 80072EFE de Windows Update.

# Características
* Reproductor compatible con los principales formatos de audio (wav, mp3, (wma, ogg y flac)
* Múltiples modos de selección de pistas, como aleatorio, reproducción aleatoria o manual para controlar la reproducción.

* Pistas especiales, como anuncios de tiempo y archivos de audio aleatorios de una carpeta.

* Un programador de eventos versátil, para archivos de audio o para iniciar/detener la reproducción, en un día y hora programados o periódicamente, como días de la semana u horas.

* Un reproductor simplificado para preescuchar archivos de audio en otra salida de sonido.

* Archivos de registro generados automáticamente para registrar cualquier actividad o error del reproductor.

# Contribuciones
¡En primer lugar, gracias por considerar contribuir a este proyecto! Salamandra es un software gratuito, pero no de código abierto; ¡esto no significa que las contribuciones no sean bienvenidas!

Siempre puedes contribuir con pruebas, documentación, traducciones, reportando errores o compartiendo ideas para nuevas funcionalidades.

Ahora mismo, lo que más necesitamos son pruebas. Todas las funcionalidades son estables en nuestros entornos de prueba, pero pueden surgir errores y comportamientos inesperados en diferentes configuraciones, sobre todo en estas primeras versiones.
